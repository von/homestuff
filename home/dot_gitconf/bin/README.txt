This directory contains extra git commands and help scripts

To create a git command:

 1) Create a file with a name in the given format git-mycommand. And this file should
    be given executable access.

 2) The created file should be included in $PATH.

 3) Then you can run the command git mycommand in any of the git repo you have.

Kudos: https://gitbetter.substack.com/p/automate-repetitive-tasks-with-custom
